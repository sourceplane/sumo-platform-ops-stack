# nextjs-cloudflare

Opinionated blueprint for teams running a pnpm and Turbo monorepo with Cloudflare Pages managed through Terraform.

## Uses

- `workspace`
- `turbo-package`
- `cloudflare-pages-turbo`
- `cloudflare-pages-terraform`

## Good fit for

- frontend-heavy platform repos
- monorepo standardization
- teams that want Pages infrastructure managed declaratively
